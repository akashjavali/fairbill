# FairBill Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a production-grade AI-powered bill auditing SaaS as a Turborepo monorepo with Next.js web, Expo mobile, and Express backend.

**Architecture:** Clean architecture monorepo — Controller→Service→Repository on backend, all external services (AI/OCR/payments) abstracted behind interfaces, shared packages for types/utils/ui/hooks/config. No business logic in frontend.

**Tech Stack:** Turborepo + pnpm, Next.js 14 (App Router), Expo (React Native), Express + TypeScript, PostgreSQL (Neon), Prisma, Zod, Vitest, Playwright, ShadCN, Tailwind, NativeWind, Razorpay, Stripe, OpenRouter

**Responsiveness rule:** Every web component must pass visual review at sm(640), md(768), lg(1024), xl(1280) breakpoints. All layouts are mobile-first.
